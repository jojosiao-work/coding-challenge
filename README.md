# README

This README would normally document whatever steps are necessary to get the
application up and running.

* Ruby version
  ruby version=3.0
* System dependencies
  preferably on a Linux Operating System
* Database creation
  database server to use is mysql
* Database initialization
  please provide credentials in the config/database.yml file.
* How to run the test suite
  RSpec is used. type `rspec` to run tests on models and controllers.
